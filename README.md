# Fundamentos_Programacion
UNA COMPILACION DE LOS EJERCICIOS REALIZADOS EN CLASE, TANTO EJEMPLOS COMO LAS ACTIVIDADES
Objetivo
Comprender el uso de las estructuras while, do-while y for, así como el comportamiento de break
y continue.
Parte 1: Ciclo while
Ejercicio 1
Escribe un programa que imprima los números del 1 al 10 utilizando while.
Ejercicio 2
El usuario ingresa números. El programa debe sumar todos los números hasta que se ingrese 0.
Ejercicio 3
El usuario ingresa números (termina con 0). Mostrar cuántos números positivos fueron ingresados,
cuaátos negativos.
Parte 2: Ciclo do-while Ejercicio 4
Reescribe el ejercicio 2 utilizando do-while.
Ejercicio 5
Escribe un programa que solicite una contraseña hasta que sea correcta. Preguntas:
¿Cuál ciclo se ejecuta al menos una vez?
¿Qué pasa si el usuario ingresa el valor correcto desde el inicio?
Sugerencia: usar la biblioteca string.h, para poder emplear las funciones strcmp
